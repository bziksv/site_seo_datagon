# Topvisor — конспект для monitoring v2

**Дата:** 2026-05-27  
**Полная карта:** [topvisor-product-map.md](./topvisor-product-map.md) · **Журнал URL:** [topvisor-pages-visited.md](./topvisor-pages-visited.md)  
**Параллель:** [cabinet-monitoring-benchmark-seranking.md](./cabinet-monitoring-benchmark-seranking.md)

Фокус: **список проектов** + **проверка позиций** — аналог Datagon `/monitoring` и `/monitoring-v2`.

---

## 1. Два уровня (как у SE Ranking)

| Уровень | URL | Что видит пользователь |
|---------|-----|------------------------|
| **Все проекты** | `/projects/` | Таблица доменов: запросы, ср. позиция, динамика, TOP 10 / 11–30 / 30+; кнопка «Проверить позиции» |
| **Один проект** | `/project/dynamics/{id}/` | Таблица ключей с датами, фильтр периода, регион (Яндекс/Google), метрики видимости |

Семантика (`/project/keywords/{id}/`) **отделена** от отчёта позиций — не смешивать в одном grid на списке проектов.

---

## 2. Дашборд `/projects/`

- **Шапка:** баланс (₽), переключатель папок («Все папки», гостевые).
- **Toolbar:** «Создать проект», настройка столбцов (модалка «Настройка столбцов»).
- **Строка проекта:** домен (favicon + имя), быстрый запуск проверки, число запросов, средняя позиция, динамика, три bucket TOP.
- **Действие:** уход в `dynamics` — основной drill-down для позиций.

**Для Datagon v2:** карточки/таблица на хабе — только дайджест; полная матрица ключ×дата — отдельный route.

---

## 3. Проверка позиций `project/dynamics/{id}/`

- **Заголовок:** `{домен} (Динамика запросов проекта) #{id}`.
- **Фильтры:** диапазон дат (hash `date1`, `date2`, `typeRange`), регион, папки/группы/теги.
- **Переключатели метрик:** «Проверка позиций», «Видимость» (средняя / Топ100).
- **Таблица:** ключи + столбцы по датам съёмов; настройка столбцов (как на projects).
- В демо-аккаунте: 1 ключ («почта»), регион Москва, Яндекс.

**Отличие от SE Ranking:** нет отдельного hash `#/detailed` — основной UI уже табличный на `dynamics`. Обзор KPI вынесен на `/projects/`, а не на `overview` внутри rankings.

---

## 4. Боковое меню проекта

Единый sidebar на всех `/project/*/{id}/` — быстрый переход в снимки, конкурентов, AI, аудит. Для monitoring v2 достаточно ссылок: **Ключи** · **Позиции** · **Настройки**.

---

## 5. Рекомендации для Datagon (сводка с SER)

| Паттерн | Topvisor | SE Ranking | Предложение для v2 |
|---------|----------|------------|---------------------|
| Список проектов | Таблица + TOP buckets | Таблица + expand регионов | Таблица или карточки с KPI; child-rows для регионов |
| Глубокий модуль | `dynamics` | `rankings#/detailed` | Отдельная страница «Позиции проекта» |
| Семантика | `keywords` | Внутри rankings / wizard | Отдельный пункт меню |
| Мульти-метрика | Видимость / TOP на dynamics | overview + detailed | Toolbar переключения метрик |
